Cuestionario de Procesos de Negocio para el Asistente de IA

Este cuestionario captura las reglas de negocio esenciales para que el asistente califique leads y tome decisiones correctas en cada etapa. Responde con claridad y, si aplica, incluye ejemplos concretos.

Sección 1

Proceso de Venta de Mayoreo

Ej.: señales de intención de compra, validaciones mínimas (volumen, frecuencia, productos de interés, datos de contacto).

Si sí, indica cómo formular las preguntas y qué nivel de detalle se requiere (SKU, categoría, cantidad).

Define qué puede responder el bot (rangos, lista base, remitir a cotización) y cuándo escalar a humano.

Incluye si el bot recopila datos previos (productos, cantidades, datos fiscales) y cuándo interviene un vendedor.

Sección 2

Proceso de Venta de Menudeo

Para clientes que compran equipo/accesorio de contado: pasos sugeridos y objetivo (visita a sucursal, pago online, reserva, etc.).

Información general que el bot puede compartir (plazos típicos, enganche, requisitos) antes de referir a sucursal.

Sección 3

Lógica de Servicios y Operaciones

Si una sucursal no ofrece el servicio (ej. reparación), ¿cómo se deriva al cliente a la sucursal más cercana que sí lo ofrece?

Términos, condiciones y plazos exactos que el bot puede responder (ventana de días, estado del producto, comprobantes, restricciones).
