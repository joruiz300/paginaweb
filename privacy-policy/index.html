Política de Privacidad – Instancia Privada de Postiz

Fecha de entrada en vigor: 18 de agosto de 2025

1. Introducción
Esta Política de Privacidad describe cómo se manejan los datos en una instancia privada y autoalojada de Postiz, asociada a este dominio para cumplir con los requisitos de las APIs de terceros. El principio fundamental es que todos los datos son controlados exclusivamente por el propietario de la instancia en su propio servidor.

2. Recopilación y Uso de Datos
La aplicación almacena credenciales de API y tokens de acceso necesarios para conectar con las plataformas de redes sociales (ej. TikTok). Esta información tiene como único propósito el funcionamiento de la aplicación para publicar y programar contenido según las instrucciones del administrador.

3. No Compartición de Datos
Dado que es una instancia autoalojada y de uso personal, ningún dato personal, credencial de API o contenido es compartido, vendido o transferido a terceros. Todos los datos residen en el servidor privado y controlado por el administrador.

4. Seguridad de los Datos
La seguridad de los datos depende de las medidas implementadas en el servidor donde se aloja la aplicación. El administrador es responsable de mantener la infraestructura segura.
